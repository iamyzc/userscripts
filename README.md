####为电影下载站添加豆瓣链接
####目前支持[mp4ba.com](http://www.mp4ba.com),[迅播影院](http://www.2tu.cc),[圣城家园](http://www.cnscg.org/)
####平时登下载网站的时候，看到一个电影，不知道好看不，还要去豆瓣电影看看评价

####进去豆瓣，还要搜索一下，很是麻烦，不如自己借助Greasemonkey自己写个脚本在网站上添加一个这个电影豆瓣链接
